# co-enterprise

Harness-optimized enterprise transformation partner with 7 sub-skills, 2 Custom Agents, and full Orchestrator routing.

- **Version**: v0.1.0
- **Sub-skills**: 7 (problem-structuring, kpi-diagnostics, initiative-prioritization, roadmap-simulation, exec-briefing, deep-research, learning-capture)
- **Custom Agents**: 2 (transformation-lead, strategy-reviewer)
- **Harness Score**: 21/21 (Expert)

## Overview

Co-Enterprise guides leadership teams through structured enterprise transformation — from problem framing through executive decision output. It decomposes ambiguous strategic challenges into decision-ready workstreams, diagnoses KPI gaps, prioritizes initiatives, simulates roadmaps, and produces concise executive briefings.

## Transformation Workflow

```
Phase 0: Problem Structuring       → co-enterprise-problem-structuring    ⏸️
Phase 1: KPI Diagnostics           → co-enterprise-kpi-diagnostics
Phase 2: Initiative Prioritization → co-enterprise-initiative-prioritization ⏸️
Phase 3: Roadmap Simulation        → co-enterprise-roadmap-simulation
Phase 4: Executive Briefing        → co-enterprise-exec-briefing          ⏸️
```
