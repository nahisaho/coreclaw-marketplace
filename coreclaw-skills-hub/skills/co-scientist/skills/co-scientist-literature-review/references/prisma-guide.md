# PRISMA Systematic Review Guide

Read this reference when conducting PRISMA-compliant systematic reviews.

## PRISMA Flow Diagram

```
Records identified (n = __)
    ↓
Duplicates removed (n = __)
    ↓
Records screened (n = __)  → Excluded (n = __)
    ↓
Full-text assessed (n = __) → Excluded with reasons (n = __)
    ↓
Studies included (n = __)
```

## 27-Item Checklist (Key Items)

| # | Item | Section |
|---|------|---------|
| 1 | Title | Identify as systematic review |
| 2 | Abstract | Structured summary |
| 6 | Eligibility | Inclusion/exclusion criteria |
| 7 | Information sources | Databases searched |
| 8 | Search strategy | Full electronic search |
| 9 | Selection process | Screening method |
| 13 | Risk of bias | Assessment tool used |
| 14 | Synthesis | How results were combined |

## Application Rules
1. Register protocol before starting (PROSPERO).
2. Search minimum 2 databases.
3. Two reviewers for screening (or document single-reviewer limitation).
4. Report PRISMA flow diagram with exact numbers at each stage.
