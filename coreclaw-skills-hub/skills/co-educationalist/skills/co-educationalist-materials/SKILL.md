# Materials Creation

Instructional materials with pedagogical scaffolding and UDL.

## Use This Skill When

- Creating worksheets, handouts, or practice materials.
- Designing slide outlines or visual aids.
- Applying UDL principles for differentiated access.

## Workflow

1. Confirm: learning objectives, target learners, material type.
2. Apply scaffolding appropriate to Bloom's level.
3. Include UDL: multiple representations, engagement options, expression modes.
4. Cite grounding theory.
5. Save to files.

## Deliverables

- `results/materials/`: generated worksheets, handouts, or slide outlines.

## Quality Gates

- [ ] Materials align with stated learning objectives.
- [ ] Bloom's cognitive level matches the target activity.
- [ ] UDL principles applied (multiple means of engagement/representation/expression).
- [ ] Theory grounding cited.

If any gate fails: identify the issue, fix, and re-validate.

## Gotchas

- ワークシートの難易度が学習目標のBloom'sレベルと一致しているか確認すること
- UDL は「全員に同じ教材」ではなく「アクセスの選択肢を提供する」こと。選択肢なしの教材は UDL 不適合
- 視覚教材はカラーユニバーサルデザインに配慮すること

## Validation Loop

1. 教材を生成
2. チェック: 目標整合性、Bloom's整合、UDL適用、理論引用
3. 不合格なら修正
4. 合格後のみ完了
