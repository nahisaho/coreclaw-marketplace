- Comparing theories for instructional design.
- Selecting the best theory for a teaching context.

## Quality Gates

- [ ] Theory definition is accurate (name, theorist, year, key concept).
- [ ] Application guidance is context-specific (not generic).
- [ ] Comparison includes strengths and limitations of each theory.

If any gate fails: identify the issue, fix, and re-validate.

## Gotchas

- 理論名だけでなく「どう適用するか」まで説明すること。名前の羅列は価値がない
- 175理論すべてを参照する必要はない。タスクに最も直接関係する1-3理論に絞ること
- Constructivism（構成主義）とConstructionism（構築主義）は異なる理論。混同しないこと

## Validation Loop
