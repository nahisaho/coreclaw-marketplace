# Competitive Benchmark

Target positioning analysis against competitors.

## Use This Skill When

- Benchmarking a target against direct competitors.
- Evaluating competitive moats and differentiation.
- Assessing market share and positioning.

## Workflow

1. Identify direct competitors (3-5) and adjacent competitors (2-3).
2. Define benchmark dimensions (market share, product, pricing, technology, team).
3. Score each dimension with evidence.
4. Identify competitive moats (or lack thereof).
5. Save comparison matrix to files.

## Deliverables

- `results/competitive-benchmark.md`: positioning analysis with comparison matrix.

## Quality Gates

- [ ] 3+ direct competitors identified with rationale.
- [ ] Benchmark dimensions relevant to investment thesis.
- [ ] Competitive moat assessment explicit (sustainable/temporary/none).
- [ ] All comparisons evidence-based (not opinion).

If any gate fails: identify the issue, fix, and re-validate.

## Gotchas

- 競合は「直接競合」だけでなく「隣接競合」（代替手段を提供する企業）も含めること
- 市場シェアの数値はソースと定義（金額ベース/数量ベース）を明記すること
- 競争優位性は「現時点」だけでなく「持続可能性」（2-3年後）も評価すること

## Validation Loop

1. 競合分析を生成
2. チェック: 競合数、ベンチマーク根拠、モート評価
3. 不合格なら修正
4. 合格後のみ完了
